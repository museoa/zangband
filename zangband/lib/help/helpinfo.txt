=== Online Help Commands===

Esc    : Leave the online help system.
?      : Return to the previous help file or menu.
Space  : Advance one page (wraps to the start at the end of the file).
Return : Advance one line (wraps to the start at the end of the file).
(-)    : Back-up half a page.
(#)    : Jump to a specific line (defaults to line zero).
(%)    : Jump to a specific file (defaults to "help.hlp").
(=)    : Hilite lines containing a string (case sensitive).
(/)    : Search for a string (use Slash + Return to continue a search).
[0..9] : Press the indicated number to activate a link.

Hint: Use Pound + Zero + Return + Slash + Return to restart a search.
